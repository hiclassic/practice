In JavaScript, objects are collections of key-value pairs, where keys are strings (or Symbols) and values can be any data type, including other objects or functions. Methods are functions that are properties of an object.

Creating Objects

Objects can be created using object literals, the `new Object()` syntax, or constructor functions.

const person = {
    name: 'Alice',
    age: 30,
    greet: function() {
        console.log(`Hello, my name is ${this.name}`);
    }    
};
