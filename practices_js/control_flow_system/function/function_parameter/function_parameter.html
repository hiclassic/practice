function_parameter

Function parameters are the variables that are passed into a function when it is called. They allow you to pass data into functions, making them more flexible and reusable.

Example


function greet(name) { 
    console.log("Hello, " + name + "!");
}
greet("Alice"); // Output: Hello, Alice!
greet("Bob");   // Output: Hello, Bob!

Explanation

In the example above, the function greet takes one parameter, name. When the function is called with different arguments ("Alice" and "Bob"), it outputs a personalized greeting.

Key Points

  • Function parameters allow you to pass data into functions.
  • You can define multiple parameters by separating them with commas.
  • Parameters can have default values if not provided during the function call.
